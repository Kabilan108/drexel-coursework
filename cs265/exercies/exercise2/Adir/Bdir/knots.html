Publishing

-i specifies the input file, and -o specifies the output file. Often, the formats can be gleaned from the file extensions. You can help Pandoc out. -f lets you specify the from formate, and -t the target format.

$ pandoc -i knots.md -o knots.html

Or,

$ pandoc -i knots.md -o knots.rtf

To publish to LaTeX or PDF, specify latex as the target format:

$ pandoc -t tex -i knots.md -o knots.pdf
